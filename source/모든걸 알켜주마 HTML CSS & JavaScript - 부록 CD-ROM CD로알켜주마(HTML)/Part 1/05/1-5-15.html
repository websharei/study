다음은 문자를 장식할 때 사용하는 태그입니다

문자 장식 태그는 참고로 아래 10가지 만 아시면 이용에 전혀 문제가 없습니다.


글자를 굵게 적고 싶을 때는 B태그를 이용 하시구요.

이탤릭체로 적고 싶은 때는 I태그를 이용하시면 됩니다.

고정폭 글자를 이용하고 싶을 때는 TT태그를 이용하세요.

위첨자의 효과는 이런거에요. 위첨자

아래첨자 효과도 보실래요? 아래첨자

글자에 밑줄을 그으실 때는 U태그를 이용하세요.
U태그는 사용자들이 링크가 걸렸다는 표시와 혼돈할 수 있으므로 조심해서 사용하세요.

아 실수, 실수를 했을때는 S태그 입니다. ^^

기존의 글자보다 크게 보이죠? 이 태그는 Big 태그 입니다.

이번에는 기존의 글자보다 작죠? 이 태그는 Small 태그 입니다.

반짝 반짝… 이 태그는 반짝 반짝하게 만들 때 사용하는 태그입니다.
근데.. 반짝거리지 않죠? 아쉽게도 넷스케이프에서만 적용될 뿐 익스플로러에서는 적용이 않된다는 군요.
