margin-left:20pt , margin-right:20pt를 사용한 예

HTML에서는 hspace(가로여백 지정) 와 vspace(세로여백 지정)를 이용하여 여백을 조정하였습니다. 하지만, Margin을 이용하면 상 하좌우를 보다 자유롭게 관리를 할 수 있습니다.
지금 보고 계신 박스는 상단(margin-top:50pt)에서 50pt , 좌측에서 20pt, 우측에서 50pt 떨어져 있습니다.
